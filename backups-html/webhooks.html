ℹ️ Formato do Payload

O webhook receberá eventos no seguinte formato:

{ "event": "message.received", "instance": "minha-instancia", "data": { "key": { "remoteJid": "5511999999999@s.whatsapp.net", "fromMe": false, "id": "ABC123..." }, "pushName": "João", "message": { "conversation": "Olá!" } } }

🎯 Eventos Disponíveis

instance.open - WhatsApp conectado
instance.close - Sessão encerrada
instance.qrcode - QR Code gerado
instance.pairing_code - Código de pareamento
message.received - Nova mensagem recebida
