## Stack
- React Native com Expo SDK 50+
- TypeScript obrigatório
- NativeWind para estilos
- React Query para data fetching

## Padrões
- Componentes em PascalCase
- Hooks customizados com prefixo "use"
- Navegação com Expo Router
- Assets otimizados (webp, max 100kb)

## Performance
- Lazy loading para telas pesadas
- FlashList ao invés de FlatList
- Memoização com useMemo/useCallback
- Imagens com cache (expo-image)

## Testes
- Jest + React Native Testing Library
- Testes E2E com Detox
- Cobertura mínima: 70%

## Build
- EAS Build para produção
- Profiles: development, preview, production
- OTA updates habilitado
